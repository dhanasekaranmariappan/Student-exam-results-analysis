# Student Exam Results Analysis

## Overview
Exploratory Data Analysis (EDA) on 1000 student exam results 
across 6 subjects using Python and Pandas.

## Key Findings
- 917/1000 students failed at least one subject
- Science has the highest failure rate (35.1%)
- English has the lowest failure rate (32.8%)
- Passing students score 112 points higher on average
- Domain insight: failing even one subject = overall failure

## Tools Used
- Python, Pandas, NumPy, Matplotlib
- Jupyter Notebook
